# Codeforces analyser statistics

### Мы делаем web-сервис, который анализирует статистику пользователя на Codeforces и предоставляет персональные рекомендации для прогресса в спортивном программировании. Пока проект не закончен и находится в постоянной доработке

## Немного про функционал сервиса

* **Детальный анализ профиля:**
Выявление сильных и слабых тем на основе истории решений пользователя. Отображаюся графики прогресса по разным темам
* **Умная рекомендация задач:**
Персональный подбор следующей задачи с учётом текущего уровня, целей пользователя и истории решённых задач.
* **AI-помощник на базе LLM:**
Помощь в разборе условий задач, объяснении разборов и ответы на вопросы, а также перевод условий задач на разные языки
