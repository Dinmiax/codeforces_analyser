# Codeforces analyser statistics

### Мы делаем web-сервис, который анализирует статистику пользователя на Codeforces и предоставляет персональные рекомендации для прогресса в спортивном программировании. Пока проект не закончен и находится в постоянной доработке

## Немного про функционал сервиса

* **Детальный анализ профиля:**

1. Визуальное отображение дней с решенными задачами
2. Графики прогресса по разным темам
3. Условия задач с кфа прямо на сайте
4. История решенных задач

* **Умная рекомендация задач:**

1. Персональный подбор задач на базе ML модели
2. Подбор случайных задач по самой плохо решаемой теме пользователя
3. Гибкий подбор задач с использованием LLM-агента

* **AI-помощник на базе LLM:**

1. Можно переводить задачи на любой язык
2. Можно узнавать интересные факты про спортивное программирование и математику
3. Оценка качества ответа LLM-агентов на основе других ответов LLM-агентов
